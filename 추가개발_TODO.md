## 1. 과목코드 ↔ 영역 ↔ 답안 자동 매칭 로직

### 현재 상태
- 과목관리에서 과목코드/과목명/정답 등록 가능
- 영역 이름을 과목명으로 매칭하여 정답 슬라이스 로드 가능
- subject_code 타입 ROI에서 코드 감지 가능

### 미구현 (TODO)
- subject_code ROI에서 감지된 코드값 → 과목관리에서 해당 코드의 과목명 조회 → subject_answer ROI의 이름에 자동 설정
- 즉, OMR에서 "01" 코드가 감지되면 → 과목관리에서 code="01"인 과목명 "국어" 찾기 → 답안 영역의 이름을 "국어"로 자동 설정 → 정답 슬라이스 자동 로드 → 채점
- 이 흐름이 일괄채점(배치)에서도 자동으로 돌아야 함

### 구현 시 고려사항
- 하나의 OMR에 과목코드 영역 1개 + 답안 영역 여러 개 → 모든 답안 영역에 동일 과목 적용
- 과목코드가 없는 OMR → 수기 매칭 (현행 유지)
- 과목코드가 과목관리에 없는 코드 → 경고 표시
- 이미지마다 과목코드가 다를 수 있음 (배치 처리 시 이미지별 독립 매칭)
