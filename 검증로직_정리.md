# OMR 마킹 감지 검증 로직 (2단계)

## 전체 파이프라인

```
[Stage 1] BFS 블롭 감지 (동그란 / 세로길쭉 분기)
   ↓
[Stage 2] sampleCell → 복합스코어 공식E/E-v4 → 1차 답안 (bubbleAnswer)
   ↓
[Stage 5] 링 4분면 교차 검증 → 2차 답안 (xvAnswer)
   ↓
[최종 판정] 1배 × 링 매트릭스 기반 결정
```

## 1배 판정 (Stage 2 — bubbleAnswer)

### 공식

| 모드 | 복합 스코어 공식 | 기본 임계값 |
|------|------------------|-------------|
| 동그란 버블 | `comp = f×0.4 + e×0.35 + qMin×0.25` | gap ≥ 0.06 OR f ≥ 0.82 |
| 세로길쭉 버블 | `comp = f×0.55 + e×0.45` (qMin 제거) | gap ≥ 0.08 OR f ≥ 0.82, AND e ≥ 0.20 |

### 추가 가드

- **백지 판별**: maxFill < 0.5 AND maxFill-minFill < 0.15 → null
- **gap 극소**: gap < 0.03 AND f < 0.92 → null (오탐 방지)
- **길쭉 백지**: maxFill < 0.35 AND maxFill-minFill < 0.10 → null

## 링 판정 (Stage 5 — xvAnswer)

### 링 영역 계산

1배 버블 박스(w×h)와 1.5배 확장 박스(w×1.5 × h×1.5) 사이의 **도넛 영역**만 샘플링.
4방향 분할: **Top / Bottom / Left / Right**.

```
┌─────────────────────┐
│     TOP (상단)      │
├───┬─────────┬───────┤
│ L │ 1배 영역 │   R   │ ← 1배 안쪽은 제외
│   │  (제외)  │       │
├───┴─────────┴───────┤
│    BOTTOM (하단)    │
└─────────────────────┘
```

### 판정 기준

- 각 버블의 **4분면 darkRatio 중 MAX**를 대표값으로
- **ringMax ≥ 0.30 AND gap ≥ 0.10** → 링 감지
- 한 방향만이라도 진하면 마킹 번짐으로 간주 (체크 표시, 부분 칠 등 대응)

## 최종 판정 매트릭스 (1배 × 링)

| 정배 (1배) | 링 (1.5배 4분면 MAX) | 플래그 | 추가 조건 | 최종 답안 | 셀 표시 |
|:---:|:---:|:---:|:---|:---:|:---:|
| `X` | `X` (동일) | `match` | - | **X** | 기본 (초록/빨강 채점색) |
| `X` | `Y` (다름) | `conflict` | 링 MAX ≥ 0.7 **AND** gap ≥ 0.3 | **Y (링 채택)** | 🟢 초록 (자동교정) |
| `X` | `Y` (다름) | `conflict` | 위 조건 미달 | **null** | 🟠 주황 별색 |
| `X` | `null` | `bubble_only` | **1배 f ≥ 0.85** (절대 임계 통과) | **X (유지)** | 🟠 주황 점선 |
| `X` | `null` | `bubble_weak` | **1배 f < 0.85** (절대 임계 미달) | **null** | 🟠 주황 별색 |
| `null` | `Y` | `xv_only` | - | **Y (링 채택)** | 🟢 초록 (자동교정) |
| `null` | `null` | `both_null` | - | **null** | 비어있음 |

## 임계값 요약

| 항목 | 값 | 의미 |
|------|-----|------|
| 1배 gap 임계 (동그란) | 0.06 | 1위-2위 복합스코어 차이 |
| 1배 gap 임계 (세로길쭉) | 0.08 | 보수적 |
| 1배 f 단독 임계 | 0.82 | gap 미달 시 대안 |
| 1배 gap 극소 방지 | gap < 0.03 AND f < 0.92 | 극단적 오탐 차단 |
| 세로길쭉 e 임계 | 0.20 | 인쇄 노이즈 배제 |
| **링 절대 임계** | 0.30 | 마킹 번짐 감지 기준 |
| **링 gap 임계** | 0.10 | 링 1위-2위 차이 |
| **링 override 임계** | MAX ≥ 0.7 AND gap ≥ 0.3 | 1배를 무시할 만큼 강한 링 신호 |
| **1배 절대 임계 (bubble_only)** | f ≥ 0.85 | 링 뒷받침 없이도 유지 가능한 수준 |

## 디자인 원칙

1. **오탐 > 미탐** — 틀린 답을 보여주는 것보다 null이 안전
2. **링 우선순위 명확** — 링이 매우 강하면 1배를 override (실제 마킹이 번진 케이스)
3. **교사 확인 유도** — 애매한 경우 별색 표시로 시각적 경고
4. **자동 교정 플래그** — 링 override / xv_only 케이스는 `_xvAutoCorrected = true` 설정 → 교정 확정 버튼 대상

## 로그 예시

```
Q6: [1] comp=0.331 | [3] comp=0.491 | [5] comp=0.591 gap=0.100 → 5
    └링4분면: [3]max=1.00(T0.40 B0.42 L0.29 R1.00) | [5]max=0.34 gap=0.344 → 3 [conflict]
    
최종: 링 매우 강함(1.00 ≥ 0.7, gap 0.344 ≥ 0.3) → [3] 채택 (초록 자동교정)
```

```
Q4: [4] comp=0.622, f=0.875 → 4
    └링4분면: [4]max=0.18 gap=0.05 → null [bubble_only]
    
최종: 1배 f=0.875 ≥ 0.85 → [4] 유지 (주황 점선 경고)
```

```
Q7: [4] comp=0.477, f=0.688 → 4
    └링4분면: [4]max=0.12 gap=0.03 → null [bubble_weak]
    
최종: 1배 f=0.688 < 0.85 → null (주황 별색)
```
