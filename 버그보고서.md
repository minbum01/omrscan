## CRITICAL (즉시 수정 필요)

### BUG-1. ROI 타입 불일치 (phone_exam vs phone)
- **파일**: scoring.js, imageManager.js
- **내용**: ROI_TYPES에서 'phone_exam'을 'exam_no'+'phone'으로 분리했지만, scoring.js와 imageManager.js에 'phone_exam' 참조가 남아있음
- **영향**: 수험번호/전화번호 감지가 일부 경로에서 실패
- **수정**: 모든 'phone_exam' → 'exam_no' || 'phone' 통일

### BUG-2. gradeResult null 접근
- **파일**: scoring.js collectData
- **내용**: `img.gradeResult.score` 접근 시 gradeResult가 null이면 에러
- **영향**: 분석만 하고 채점 안 한 이미지에서 크래시
- **수정**: `img.gradeResult?.score || 0` 으로 변경

### BUG-3. 0으로 나누기 (percentage 계산)
- **파일**: scoring.js 개인별 성적표 렌더
- **내용**: `(s.score / max) * 100` 에서 max가 0일 때 Infinity
- **영향**: 만점이 0인 과목(미설정)에서 NaN/Infinity 표시
- **수정**: `max > 0 ? (score / max) * 100 : 0`

## HIGH (기능 오류)

### BUG-4. 과목별 점수 계산 오류
- **파일**: scoring.js collectData
- **내용**: 과목별 score를 gradeResult.details에서 추출하는 로직이 불안정
- **영향**: 다과목 시 과목별 점수가 정확하지 않을 수 있음
- **원인**: details의 score 필드가 채점 방식에 따라 없을 수 있음
- **수정**: gradeResult에서 직접 과목별 점수 계산 대신, ROI별 정답 수 × 배점으로 계산

### BUG-5. 세션 로드 시 교시(period) 지원
- **파일**: session.js, app.js
- **내용**: App._initPeriods() 호출이 집에서 추가된 코드에 있지만, 여기에서는 미구현
- **영향**: 교시 기능 사용 시 세션 로드 실패 가능
- **수정**: App._initPeriods가 없으면 무시하도록 가드 추가

### BUG-6. 일괄채점 시 과목 정답 로드 누락
- **파일**: batch.js
- **내용**: batch에서 채점 시 UI._loadAnswersFromSubject 호출하지만, 다과목 환경에서 모든 ROI에 대해 호출되는지 불확실
- **영향**: 일괄채점 시 일부 과목 정답 미적용
- **수정**: 채점 전 모든 subject_answer ROI에 대해 과목 정답 로드 보장

## 추가 점검 결과 (사용자 피드백 + 채점/교정 로직)

### BUG-13. 전체 진하기 체크박스 풀림 [사용자 제보]
- **파일**: canvas.js initAdjustBar (576~589행)
- **내용**: '전체' 체크 시 change 이벤트로 처리 → 처리 완료 후 `e.target.checked = false` 하는데, DOM 동기화가 안 되어 체크가 풀린 것처럼 보임
- **수정**: setTimeout으로 DOM 동기화 보장, 또는 체크박스 대신 버튼 방식으로 변경

### BUG-14. 박스 이동 시 간헐적 오류 [사용자 제보]
- **파일**: canvas.js handleMove (245~250행)
- **내용**: ROI 드래그 중 selectedRoiIdx가 유효하지 않으면 `roi` undefined → 크래시
- **원인**: 드래그 도중 ROI 삭제되거나 rois 배열 변경 시 발생
- **수정**: `if (!roi || !orig) return;` null 체크 추가

### BUG-15. 양식 만들 때 과목 연결 불가 [사용자 제보]
- **파일**: ui.js ROI 설정
- **내용**: 과목관리에 과목이 없는 상태에서 박스를 만들면 과목명 지정 불가
- **수정**: 빈 이름이면 [과목1], [과목2] 자동 부여 → 나중에 과목관리와 매칭

### BUG-16. xlsx 지원 없음 [사용자 제보]
- **파일**: subjectManager.js, scoring.js
- **내용**: 현재 CSV만 지원, xlsx 불러오기/내보내기 없음
- **수정**: SheetJS(xlsx) 라이브러리 추가하여 xlsx 읽기/쓰기 지원

### BUG-17. handleEnd에서 imgObj null 체크 누락
- **파일**: canvas.js handleEnd (334~350행)
- **내용**: ROI 이동 끝날 때 `App.getCurrentImage()`가 null 반환 가능
- **수정**: imgObj null 체크 추가

### BUG-18. Grading.grade() 파라미터 누락
- **파일**: grading.js (149행)
- **내용**: 정답 저장 후 재채점 시 `this.grade(img.results)` — imgObj 파라미터 누락
- **영향**: getAnswersForRoi에서 roi 정보를 못 가져와 채점 실패
- **수정**: `this.grade(img.results, img)` 으로 변경

### BUG-19. 수기 교정 후 리렌더 시 상태 소실 가능
- **파일**: ui.js applyCorrection (1680~1703행)

### BUG-20. 진하기 전체 적용 이벤트 충돌
- **파일**: canvas.js (576~589행)
