# 게시용 전체 성적표 기획안

## 개요
각 세션의 교정 확정된 OMR 결과를 기반으로, **편집 가능한 HTML 성적표**를 생성.
사용자가 HTML 안에서 직접 수정(헤더, 색상, 과목 합산 등)할 수 있고,
나중에 세션 간 통합을 위해 **CSV로도 결과 데이터를 내보내기**.

---

## 1. 데이터 저장 (CSV)

### 목적
- 프로그램 부하 없이 세션별 결과를 파일로 보관
- 나중에 여러 세션을 외부에서 통합할 때 사용

### 저장 시점
- **교정 확정** 완료 후 자동 저장 (세션 폴더 내)
- 또는 채점및결과 탭에서 "결과 CSV 내보내기" 버튼

### 파일 구조
세션 폴더 내: `{세션명}_결과.csv`

```
수험번호,이름,생년월일,전화번호,과목1_점수,과목1_맞은수,과목1_1번,과목1_2번,...,과목2_점수,...
7014490,홍길동,990101,01012345678,85,17,3,5,4,2,...,90,18,1,3,2,...
```

- 헤더: 학생 정보 + 과목별(점수, 맞은수, 문항별 마킹값)
- 1행 = 1학생
- 문항별 마킹값: 숫자(선택지) 또는 빈칸(미응답)

## 2. HTML 성적표

### 생성 방식
- 채점및결과 탭에서 **"게시용 성적표 생성"** 버튼
- 세션 폴더 내에 `{세션명}_성적표.html` 파일 생성
- 생성 후 기본 브라우저에서 자동 열기

### HTML 특징
- **독립 실행형** — 프로그램 없이 브라우저만으로 열람/편집 가능
- **인라인 CSS + 인라인 데이터** — 외부 의존성 없음
- **편집 가능** — `contenteditable` 속성으로 텍스트 직접 수정
- **인쇄 최적화** — `@media print` CSS로 A4 출력 대응

---

### 2-1. 성적표 구성 (위→아래)

#### A. 헤더 영역
| 항목 | 데이터 소스 | 편집 가능 |
|------|-----------|----------|
| 시험명 | 세션명 (SessionManager.currentExamName) | ✅ 클릭하여 수정 |
| 날짜 | 세션 시험일자 (currentExamDate) | ✅ |
| 교수명/출제자 | 빈칸 (사용자 직접 입력) | ✅ |
| 과목 | 과목 목록 (SubjectManager) | ✅ |
| 응시인원 | 자동 계산 (OMR 인식 완료 인원) | 자동 |
| 평균점수 | 자동 계산 | 자동 |
| 평균맞은문항수 | 자동 계산 | 자동 |

#### B. 과목 선택 (체크박스)
- 각 과목별 체크박스 → 체크한 과목끼리 합산
- 체크 변경 시 **점수 분포, 문항분석 자동 재계산** (JavaScript)
- 기본값: 전체 과목 체크

#### C. 점수 분포표 + 그래프
| 항목 | 설명 |
|------|------|
| 점수 구간 | 100점, 95점, 90점, ... 5점 단위 |
| 인원수 | 해당 구간에 속하는 학생 수 |
| 막대그래프 | Canvas 또는 SVG로 시각화 |

#### D. 문항별 정답분석표
| 열 | 설명 | 데이터 소스 |
|-----|------|-----------|
| 문항 | 문항번호 | ROI startNum 기준 |
| 정답 | 정답 번호 | SubjectManager 정답 |
| 정답률 (전체) | 전체 학생 정답률 % | calcItemAnalysis |
| 정답률 (상위 4%) | 상위 4% 그룹 정답률 | calcItemAnalysis (upper) |
| 정답률 (상위 10%) | 상위 10% 그룹 정답률 | calcItemAnalysis |
| 선택지별 비율 | 1~5번 각각 선택 비율 % | calcItemAnalysis (distTotal) |

- 정답률 낮은 문항: 빨간 배경
- 정답률 높은 문항: 초록 배경
- 색상 기준값은 사용자가 조정 가능

#### E. 정답률 그래프
- 문항별 막대(평균 정답률) + 선(상위4%, 상위10%)
- SVG 또는 Canvas

#### F. 시험 정보 영역 (편집 가능)
| 항목 | 기본값 | 편집 |
|------|--------|------|
| 문항수 | 자동 계산 | ✅ |
| 과목별 문항수 | 자동 계산 | ✅ |
| 메모/코멘트 | 빈칸 | ✅ 자유 입력 |

- 사용자가 원하면 킬러문항, 필수문항 등 직접 타이핑 가능

#### G. 최고 오답률 영역
| 항목 | 설명 |
|------|------|
| 오답률 상위 5문항 | 자동 계산 (정답률 낮은 순) |
| 각 문항의 오답 선택지 분포 | 어떤 오답이 가장 많이 선택됐는지 |

- 자동 계산 + 사용자 코멘트 입력 가능

---

### 2-2. HTML 내 JavaScript 기능

```
1. 과목 체크박스 → 합산 재계산
   - 점수 분포 재계산
   - 문항분석 테이블 필터링
   - 그래프 재렌더링

2. contenteditable 영역
   - 헤더 텍스트 클릭 → 직접 수정
   - 시험 정보 영역 자유 입력

3. 인쇄 버튼
   - window.print() 호출
   - @media print로 깔끔한 A4 출력

4. 색상 설정 (선택)
   - 정답률 기준값 슬라이더
   - 배경색 커스터마이즈
```

## 3. 구현 순서

### Step 1: 결과 CSV 저장
- 교정 확정 후 세션 폴더에 CSV 자동 저장
- `Scoring.collectData()` 결과를 CSV 형식으로 변환

### Step 2: HTML 템플릿 생성
- `scoring.js`에 `generatePublicReport()` 함수 추가
- 데이터를 인라인 JSON으로 HTML에 삽입
- HTML 내 JavaScript가 JSON을 읽어서 테이블/그래프 렌더링

### Step 3: 과목 합산 + 편집 기능
- 체크박스 이벤트 → 재계산 로직 (HTML 내 JS)
- contenteditable 영역

### Step 4: 그래프
- SVG 기반 점수 분포 막대그래프
- SVG 기반 문항별 정답률 그래프

### Step 5: Electron 연동
- "게시용 성적표 생성" 버튼 → HTML 파일 저장 → 브라우저 열기
- 세션 폴더 내 저장

## 4. 확인 사항 (확정)

### A1. 점수 구간
- 기본: **5점 단위** (100, 95, 90, ..., 0)
- HTML 내에서 사용자가 **구간 단위 변경 가능** (5점/10점/20점 등 드롭다운)
- 데이터 레이블 추가/제거 토글
- 차트 색상, 표시 형식 등 조작 가능

### A2. 상위 그룹 비율
- **상위 12%** (합격선 기준) + **상위 30%**
- 문항별 정답률 그래프에서 3개 선: 전체 평균, 상위 12%, 상위 30%

### A3. 과목 합산 시 문항번호
- **과목별 분리 표시** (국어 1~20, 영어 1~20 각각)
- 합산 점수는 상단 점수 분포에만 반영
- 문항분석은 과목별 독립 섹션

### A4. 학생별 점수 목록
- 포함 (수험번호, 이름, 과목별 점수, 총점, 석차)
- HTML 내에서 **마스킹 버튼** — 클릭 시 수험번호/이름 일부를 *** 처리
- 마스킹 해제도 가능 (토글)

### A5. 저장 경로
- **각 세션 폴더 내** 고정 저장
- 파일명: `{세션명}_성적표.html`
