# 다과목 채점 기획서 (확정)

## 핵심 원칙
- **반드시 한 세션 안에서** 모든 과목 처리
- 각 과목은 **1번부터** 시작 (연속 번호 아님)
- 같은 과목명의 ROI는 합산 (국어 1-10 + 국어 11-20 = 국어 총점)

---

## 케이스 1: 한 OMR 용지에 여러 과목

**예시**: 공무원 시험 — OMR 1장에 국어 + 영어 + 한국사

```
[OMR 1장 - 학생1]
├── 영역 "국어"   startNum=1   numQ=25   → 국어 1~25번
├── 영역 "영어"   startNum=1   numQ=25   → 영어 1~25번
└── 영역 "한국사" startNum=1   numQ=25   → 한국사 1~25번

각 과목 1번부터 독립 시작
```

**과목 판별**: ROI 영역 이름(settings.name)으로 자동 분류

---

## 케이스 2: 과목별 OMR이 따로, 같은 세션에서 처리

**예시**: 로스쿨 — 언어이해 OMR + 추리논증 OMR 별도

```
[같은 세션에 60장 업로드]
├── 학생1_언어이해.jpg  → ROI "언어이해" 1~40번
├── 학생2_언어이해.jpg  → ROI "언어이해" 1~40번
├── ...
├── 학생1_추리논증.jpg  → ROI "추리논증" 1~40번
├── 학생2_추리논증.jpg  → ROI "추리논증" 1~40번
└── ...
```

**과목 판별**: ROI 영역 이름으로 자동 분류 (나중에 과목코드 연동 추가)
**학생 매칭**: 수험번호(우선) 또는 전화번호로 같은 학생의 다른 OMR 연결

---

## 같은 과목 합산

ROI 이름이 같으면 같은 과목으로 합산:

```
영역 "국어" startNum=1  numQ=10  → 32점
영역 "국어" startNum=11 numQ=10  → 40점
영역 "국어" startNum=21 numQ=5   → 20점
───────────────────────────────────
국어 총점: 92점
```

## OMR 결과표

### UI
- 상단에 **과목 선택 드롭다운** (국어 / 영어 / 한국사)
- 선택한 과목의 마킹 내용 + 정오표만 표시
- 점수는 해당 과목 점수

### CSV 다운로드
- **현재 과목 다운로드**: 보고 있는 과목 1개 파일
- **전체 과목 다운로드**: 과목 수만큼 파일 생성 (국어.csv, 영어.csv, 한국사.csv)

---

## 성적일람표

### 테이블 구조

```
응시번호 | 성명 | 생년월일 | ── 국어 ─────────────── | ── 영어 ─────────────── | ── 총점 ──────────────
                          | 맞은수 점수 표준점수 석차 백분위 | 맞은수 점수 표준점수 석차 백분위 | 맞은수 점수 표준점수 석차 백분위
```

- 응시자 정보: 메인 (좌측 고정)
- 과목별: 사용자가 뱃지로 지정한 성적 열이 **과목 수만큼 반복**
- 총점: 전체 합산에 대해서도 동일 열 반복
- 과목 수 제한 없음 (가로 스크롤)

### 과목별 석차/표준점수/백분위
- 국어 석차 = 국어 점수 기준 순위
- 영어 석차 = 영어 점수 기준 순위
- 총점 석차 = 총점 기준 순위
- 각각 독립 계산

---

## 문항분석표

### 과목별 분석
- 상단에 **과목 선택 드롭다운**
- 선택한 과목의 문항만 분석 (정답률, 변별도, 반응분포)
- 상위/하위 27% 그룹은 **해당 과목 점수 기준** (총점 아님)

### CSV 다운로드
- 현재 선택 과목 다운로드
- 전체 과목 다운로드 (과목별 파일)

---

## 동명이인 / 체킹 오류 감지

### 조건
- 이름이 같은데 수험번호가 다른 경우

### 표시
- **OMR 결과표 + 성적일람표** 양쪽에서 해당 행 노란 별색
- 툴팁: "동명이인 또는 체킹 오류 확인 필요"

---

## 구현 순서

```
Step 1: collectData 재설계 — 과목별 분리 + 학생 합산
Step 2: OMR 결과표 — 과목 드롭다운 + 과목별 표시
Step 3: 성적일람표 — 과목별 열 반복 + 총점 열
Step 4: 문항분석표 — 과목 드롭다운 + 과목별 분석
Step 5: CSV 다운로드 — 현재 과목 / 전체 과목
Step 6: 동명이인 감지 + 별색
Step 7: (나중에) 과목코드 연동
```
